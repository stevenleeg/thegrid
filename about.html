about

the developer(s)

The Grid is a game designed and developed by Steve Gattuso. At this point everything you see has been created by me.

Dedicated server hardware has also been graciously donated by James Anderson.

technical stuff

This game was originally a fun side project for a simple idea I had. Its main goal was to help me learn about emerging web technologies (HTML5, CSS3, and WebSockets). For this reason, the game is mostly incompatible with any non-recent browsers.

The server side is written using Python and the wonderful realtime library: Tornado. It also uses Redis for a persistant database backend.
